# HTML-Learning

Welcome to my HTML practice and learning repository.

This repo contains topic-wise tutorials and examples to help understand core HTML concepts. Each file focuses on a specific topic with practical examples.

## 📁 Files

- **Topic01-basic-tag-practice.html** – Introduction to HTML, basic tags, and page structure  
- **Topic02-image-tag-practice.html** – Working with `<img>` tag to embed images  
- **Topic03-image-tag-practice.html** – More practice with image attributes (alt, width, height)  
- **Topic04-anchor-tag-practice.html** – Creating hyperlinks using `<a>` tag  
- **Topic05-text-formatting-tag-practice.html** – Text formatting tags like `<b>`, `<i>`, `<u>`, etc.  
- **Topic06-pre-tag-practice.html** – Displaying preformatted text using `<pre>` tag  
- **Topic07-big-small-tag-practice.html** – Using `<big>` and `<small>` for text sizing  
- **Topic08-superscript-subscript-tag-practice.html** – Using `<sup>` and `<sub>` for math/scientific notation  
- **Topic09-semantic-tag-practice.html** – Semantic HTML tags like `<header>`, `<footer>`, `<article>`, etc.  
- **Topic10-ordered-list-tag-practice.html** – Creating numbered lists with `<ol>`  
- **Topic11-unordered-list-tag-practice.html** – Creating bullet lists with `<ul>`  
- **Topic12-table-tag-practice.html** – Structuring data in tables using `<table>`, `<tr>`, `<td>`, etc.  
- **Topic13-form-tag-practice.html** – Building HTML forms with inputs, labels, buttons, etc.  
- **Topic14-iframe-tag-practice.html** – Embedding external pages using `<iframe>` tag
